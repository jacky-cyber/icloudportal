欢迎使用iCloudPortal!
========================================

*iCloudPortal是什么？*
iCloudPortal是您和您的组织高效工作智慧办公的云门户。拥有工作台、人事管理、考勤打卡、流程审批、任务协作督办、合同管理、知识中心、便签、内部云盘、内部即时通讯、日报周报、工作日历等主要功能。

----------------------------------

* iCloudPortal多终端自适应，支持手机、Pad、PC等多种终端，以便用户可以随时随地办公和处理业务。
* 流程可以在运行时随时定制，包括增加、删除和修改，并即时生效。流程图能够视化编辑；流程表单也可以自定义内容、类型、表现风格、计算逻辑等。
* 支持持微信服务号接入：通过服务号方便免费信息推送到微信，提醒用户处理相关业务。微信可作为程序入口，方便用户使用。
* 拥有灵活的计算器：例如，人事工资子系统中的各个薪资项目可以自定义，并执行脚本计算工资，以应对未来薪资处理逻辑的不断变化。
